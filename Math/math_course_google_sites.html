课程特色

🧮

系统化课程设计

5个级别，由浅入深，涵盖计算、图形、应用题与逻辑推理，帮助孩子构建完整的数学知识体系。

🧩

思维训练为核心

注重数学思维与解题策略的培养，不只是做题，更是理解"为什么"，锻炼逻辑分析能力。

🚀

暑期衔接与提升

巩固已学知识，预习新学期内容，利用暑假查漏补缺、拓展提高，让孩子开学更自信。

课程介绍

Level 1
🎯 适合学生
  • 已掌握基本加减法，准备学习三位数运算和初步逻辑推理的学生
  • 希望在暑假巩固计算基础，同时接触图形与数据分析入门的学生
📖 教学内容
三位数加减法、运算律与计算策略、排队问题与年龄谜题、平面与立体图形、条形图与计数表
Session 1–2：三位数运算、运算律、加减法策略
Session 3–4：逻辑推理、图形识别、数据与方向
Level 2
🎯 适合学生
  • 已掌握基本乘法运算，准备深入学习乘除法与运算规律的学生
  • 希望在暑假拓展数学思维，接触括号运算、分配律和逻辑推理题型的学生
📖 教学内容
奇偶数规律、竖式乘法、括号运算、分配律、乘除10/100/1000、余数除法、幻方与规律探索
Session 1–2：奇偶数、竖式乘法、括号与分配律
Session 3–4：除法深化、幻方、页码与时间应用题
Level 3
🎯 适合学生
  • 已掌握四则运算基础，准备系统学习分数运算和运算顺序的学生
  • 希望在暑假提升分数计算能力，并接触条形图应用与经典逻辑题型的学生
📖 教学内容
运算顺序、括号运用、GCF/LCM、分数加减乘法、条形图应用题、鸡兔同笼
Session 1–2：运算顺序与括号、分数基础（GCF/LCM、加减）
Session 3–4：分数乘法与应用、条形图、鸡兔同笼
Level 4
🎯 适合学生
  • 已掌握分数基本运算，准备学习小数运算、百分比和高级分数技巧的学生
  • 希望在暑假系统提升数与运算能力，为初中数学做准备的学生
📖 教学内容
质数与合数、质因数分解、小数四则运算、分数与小数互转、分数计算技巧、百分比、平均数
Session 1–2：质数与因数分解、小数运算与分数互转
Session 3–4：分数计算技巧、百分比与平均数应用
Level 5
🎯 适合学生
  • 已掌握小学阶段数学知识，准备系统学习初中代数与几何的学生
  • 希望在暑假提前接触负数运算、方程、不等式和几何计算的学生
📖 教学内容
负数与指数运算、实数性质、一元方程与不等式、平行线与三角形、圆与扇形、立体几何体积与表面积
Session 1–2：数的运算基础（负数、分数、百分数、指数）、代数核心
Session 3–4：平面几何（平行线、三角形、圆、扇形）、立体几何
