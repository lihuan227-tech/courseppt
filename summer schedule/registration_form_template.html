Google Form 搭建指南 Build Guide:

1. 打开 forms.google.com → 新建表单
2. 设置表单标题和描述（同上方 header）
3. Section 1: Short Answer × 4（姓名、email、电话、学生姓名）+ Multiple Choice（年级）→ 全部 Required
4. Section 2: "Add section" → Multiple Choice（校区）→ Required
5. Section 3: 新分区 → Checkboxes（可多选 Session）→ Required
6. Section 4: 新分区 → 三个 Multiple Choice（中文/英语/数学），每个包含 "不选 N/A"
7. Section 5: 新分区 → 学费参考表（用 Description 文本展示）+ Short Answer "确认应付总额"
8. Section 6: 新分区 → 退费政策（用 Description 文本展示退费条款）+ Required Multiple Choice "是否同意"
9. Section 7: 课表日历（仅在网页版模板中展示，Google Form 无法嵌入交互日历，可用 Description 提示家长参考网站课表）
10. Section 8: 新分区 → Zelle 付款信息（用 Description 展示 gredu2019@gmail.com）+ Multiple Choice（已付款/稍后付款）+ Short Answer（付款人姓名、备注）
11. Settings → 勾选 "Collect email addresses" 和 "Send responders a copy"
